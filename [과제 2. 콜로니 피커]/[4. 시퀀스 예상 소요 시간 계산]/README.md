## 1. 목표
로봇 시퀀스를 실제로 실행하지 않고,
**미리 전체 작업 시간을 계산**하는 것이 목적이다.

---

## 2. 기본 아이디어
시퀀스를 여러 개의 **구간(segment)**으로 나누고,
각 구간의 시간을 계산해서 합산한다.

---

## 3. 시퀀스 분해
- 이동 구간: XYZ 이동, 직선 이동
- 비이동 구간: 대기(dwell), 그리퍼 동작, 흡착 안정화 등

---

## 4. 이동 구간 시간 계산 (XYZ 로봇 기준)
1. 축별 이동 거리 계산:
   - ΔX, ΔY, ΔZ
2. 각 축에 대해:
   - 최대 속도 V
   - 최대 가속도 A
3. 사다리꼴 속도 프로파일 기준으로 축별 이동 시간 계산
4. **동시에 움직이는 축 중 가장 긴 시간이 해당 구간의 시간**
   - T_move = max(Tx, Ty, Tz)

---

## 5. 직선 이동(MoveL)의 경우
- 경로 길이 L = √(ΔX² + ΔY² + ΔZ²)
- TCP 속도/가속 제한으로 이동 시간 계산

---

## 6. 비이동 시간 처리
- dwell(0.2s) → 그대로 더함
- 그리퍼/흡착 → 장비 스펙 또는 실측값 사용
- 카메라 촬영/처리 → 고정 시간으로 모델링

---

## 7. 전체 시간 계산
- 전체 시간 = Σ(각 구간 이동 시간) + Σ(대기/IO 시간)

---

## 8. 실무적 장점
- 사이클 타임 예측 가능
- 생산성 평가에 활용 가능
- 실제 실행 전 안전한 상한 시간 산출 가능
