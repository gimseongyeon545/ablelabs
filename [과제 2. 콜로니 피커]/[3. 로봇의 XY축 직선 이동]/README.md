## 1. 핵심 개념
- 로봇의 관절 움직임과 무관하게,  
- **엔드이펙터의 XY 경로가 베이스 좌표계에서 직선이면 된다.**

</br>

## 2. 직선 경로 정의
- 시작점: P₁ = (X₁, Y₁, Z)
- 목표점: P₂ = (X₂, Y₂, Z)
- XY 평면에서 두 점을 잇는 **직선 경로**를 기준으로 한다.

</br>

## 3. 직선 보간 방식
1. 경로를 파라미터 t로 표현:
   - X(t) = X₁ + t(X₂ − X₁)
   - Y(t) = Y₁ + t(Y₂ − Y₁)
2. t를 0 → 1까지 일정 간격으로 나누어 waypoint 생성
   - 예: 1~2 mm 간격

</br>

## 4. 로봇이 직선을 유지하는 방법
- 각 waypoint에 대해 **역기구학(IK)**을 계산
- 이전 관절 해를 seed로 사용하여 연속성 유지
- 결과적으로 TCP가 직선 궤적을 따라 이동

</br>

## 5. 속도와 안정성
- trapezoidal 또는 S-curve 속도 프로파일 적용
- 급가속/급정지를 피해서 흔들림 방지

</br>

## 6. 실제 작업 흐름
1. Z_safe에서 XY 직선 이동
2. 목표 위치 도달 후 Z_work로 하강
3. 작업 수행

</br>

## 7. 장점
- 충돌 위험 감소
- 경로 예측 가능
- 반복 정밀도 향상
